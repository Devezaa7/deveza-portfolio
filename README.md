# Guilhermy Deveza — Portfolio 🚀

Fala! Esse é meu portfólio pessoal, construído do zero pra mostrar meus projetos como Full Stack Developer, sem enrolação e sem cara de template genérico.

**Veja ao vivo:** https://deveza-portfolio.vercel.app/

Curtiu o resultado? Sinta-se à vontade pra dar uma olhada no código, e se quiser trocar uma ideia sobre um projeto, é só chamar em deveezag@gmail.com 😊

## O que tem aqui

* **Design escuro e premium 🌑**: fundo quase preto, tipografia grande, azul só como assinatura em detalhes — nada de excesso visual.
* **Animações discretas ✨**: reveals suaves ao rolar a página, com Framer Motion. Nada de efeito exagerado, só o necessário pra dar vida.
* **Projetos reais 💻**: cada card leva direto pro repositório e pro deploy — não é só descrição, dá pra testar.
* **Responsivo de verdade 📱**: testado em mobile, tablet e desktop.
* **Copiar e-mail com um clique 📋**: sem formulário de contato chato, só um botão rápido.
* **Performance em primeiro lugar ⚡**: imagens otimizadas automaticamente pelo Next.js, fontes carregadas de forma eficiente.

## Tecnologias usadas

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS v4
* Framer Motion
* Componentes no padrão shadcn/ui

## Rodando local

```bash
git clone https://github.com/Devezaa7/deveza-portfolio.git
cd deveza-portfolio
npm install
npm run dev
```

Abre [localhost:3000](http://localhost:3000) e pronto.

## Por que esse portfólio é diferente

* **Foco no que importa**: sem seções genéricas de "certificações" ou "hobbies" — só os projetos e o essencial sobre mim.
* **Atenção aos detalhes**: cada micro-interação (hover, scroll, copiar e-mail) foi pensada, não é só CSS de template.
* **Projetos de verdade**: sistemas completos com backend, autenticação e banco de dados — não só landing pages bonitinhas.
